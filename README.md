# ✨ Smart JS Calculator

A sleek, modern, and fully responsive web-based calculator built using **JavaScript**, **Bootstrap 5**, and custom **CSS3**. This project features a clean dark-themed user interface with smooth glowing animations and is optimized to look pristine on all screen sizes—from small mobile devices to large desktop monitors.

## 📱 Features

* **Fully Responsive UI:** Smoothly adapts to Mobile, Tablet, Laptop, and 4K displays using custom CSS grid and Bootstrap media queries.
* **Smart Operator Handling:** Prevents consecutive operators (e.g., `++`, `**`) to ensure error-free calculations.
* **Premium Visuals:** Clean dark theme background with vibrant accent colors for action buttons (AC, Clear, Operators, and Equal).
* **Glow Animation:** Beautiful text-shadow glowing effect in the footer section.
* **Advanced Layout:** Leverages CSS Grid for precise button alignment and fluid responsiveness.
